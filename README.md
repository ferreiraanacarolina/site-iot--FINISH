# Portfólio de Conclusão – Técnico em IoT (Senac)

## Descrição  
Este site é meu portfólio pessoal criado para a conclusão do curso técnico em Internet das Coisas (IoT) no Senac. O objetivo é mostrar os conhecimentos adquiridos ao longo dos 3 anos do curso e servir como parte da avaliação final para a conclusão.

## Funcionalidades  
- Formulário de contato em modal (pop-up)  
- Sistema de votação para matérias do curso  
- Apresentação dos projetos desenvolvidos durante o curso (ex: urna eletrônica, automação, etc.)  
- Links para documentação, simulações e vídeos dos projetos  

## Como executar  
1. Descompacte a pasta do projeto.  
2. Abra o arquivo `index_modelo1.html` (ou equivalente).  
3. Navegue pelo site normalmente no seu navegador.
